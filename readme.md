# Just snow
It's just snow

## Usage

```html
<script src="https://cdn.jsdelivr.net/npm/just-snow/dist/just-snow.min.js"></script>
```

If you want to connect it in <head>

```html
<script src="https://cdn.jsdelivr.net/npm/just-snow/dist/just-snow.min.js" async defer></script>
```
